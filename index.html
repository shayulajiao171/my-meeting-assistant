Quick Guide

3 分钟跑通会议闭环

最快体验路径

填入示例 → 生成并保存会议 → 会后跟进填入示例 → 生成会议纪要 → 任务追踪查看 Action Items。

1. 会前准备

填写基础信息或使用随机示例，点击“生成并保存会议”，系统会生成会前简报并自动保存。

2. 会中开会

仍在腾讯会议、飞书或 Zoom 中开会，结束后把转写文本或手动纪要粘贴回来。

3. 会后整理

进入“会后跟进”粘贴记录，AI 会提取会议摘要、关键结论和待办事项。

4. 跟进待办

Action Items 会同步到“任务追踪”，可更新负责人、截止时间和状态。

工具边界 不做实时视频会议，只处理会前准备和会后转写内容。 数据说明 示例为随机办公会议，本地数据仅保存在当前浏览器。
一句话理解： 它是“会前准备秘书 + 会后纪要秘书”，把会议变成可准备、可记录、可追踪的协作流程。
