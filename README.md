# 视频会议助手 Demo

一款轻量级 AI 会议效率工具 Demo，聚焦办公视频会议中的会前准备与会后跟进。

## 核心功能

- 会议看板：查看会议状态、待准备会议和待跟进任务。
- 会前准备：输入会议主题、目标、决策方式、会议节奏、参会角色和会议上下文，生成结构化会前简报。
- 会后跟进：粘贴会议记录，生成会议纪要、关键结论、风险提醒和待办事项。
- 任务追踪：按状态管理 Action Items，识别今日到期、即将到期和逾期任务，并生成可复制提醒消息。
- 会议管理：支持会议搜索、删除、归档和闭环状态展示。
- 跟进消息：根据会议结论和待办事项生成可复制的会后同步消息。

## 本地运行

```bash
node server.js
```

然后在浏览器打开：

```text
http://localhost:5173
```

## 接入真实 DeepSeek

项目通过本地代理接口调用 `deepseek-chat`，不要把 API Key 写进前端页面。

1. 复制 `.env.example` 为 `.env`
2. 在 `.env` 中填写 `DEEPSEEK_API_KEY`
3. 运行 `node server.js`

如果没有配置 API Key，页面会自动降级为本地模拟 AI 输出，方便稳定演示。

## 技术说明

当前版本为纯前端静态 Demo：

- `index.html`：页面结构
- `styles.css`：视觉样式与响应式布局
- `app.js`：本地状态、模拟 AI 输出、任务管理和复制交互
- `server.js`：本地静态服务与 DeepSeek 代理接口

AI 生成优先调用 DeepSeek；接口不可用时自动使用本地模拟逻辑，保证产品流程不断。
